To use the PololuMaestro plugin, there are some setup steps:

-Download and install:
"Maestro Servo Controller Windows Drivers and Software"
from:
https://www.pololu.com/product/1350/resources

-Open the Maestro Control Center software. 

-Select your Pololu board from the "Connected to" drop menu.
-Set Serial mode to:
USB Dual Port

-Click 'Apply Settings' (button in the lower right corner

-Close the control center

This only has to be done once for each Pololu board - the preference is stored on the board's EEPROM memory.
